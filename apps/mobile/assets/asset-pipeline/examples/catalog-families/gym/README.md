## Intencao de composicao

equipamento urbano compacto com fachada ativa e leitura de treino de bairro

## Modos de implementacao

- `new-family-needed`

## Deve capturar

- entrada frontal legivel
- faixa ou toldo funcional
- volumetria compacta

## Deve evitar

- caixa neutra sem identidade
- hospital
- predio institucional grande
