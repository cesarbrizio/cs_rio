# Mercado clandestino / ponto escondido

Familia visual: `market-clandestino`

Categoria: `poor`
Style guide: `default`
Kinds cobertos: `mercado-negro`

## Intencao de composicao

ponto discreto, mais aberto que a boca, mas ainda incrustado em tecido pobre

## Modos de implementacao

- `new-family-needed`

## Deve capturar

- fachada discreta
- ambiente popular
- volumetria pequena ou média

## Deve evitar

- loja formal demais
- armazem industrial pesado
- quiosque neutro
