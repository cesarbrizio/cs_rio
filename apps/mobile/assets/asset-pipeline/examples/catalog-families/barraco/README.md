# Barracos individuais e construcoes improvisadas

Familia visual: `barraco`

Categoria: `favela`
Style guide: `dense-favela`
Kinds cobertos: `barraco-1`, `barraco-2`, `barraco-3`, `barraco-4`, `barraco-5`

## Intencao de composicao

massa vertical improvisada, com anexos, laje, telha ondulada e fachada remendada

## Modos de implementacao

- `existing-family`

## Deve capturar

- volumetria irregular
- mistura de reboco e tijolo aparente
- escada ou anexo lateral
- cobertura improvisada

## Deve evitar

- casinha de desenho
- icone geométrico limpo
- fachada institucional
